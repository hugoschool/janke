# JankE

Extension that adds Epitech specific features to Jenkins

## Features

- GitHub Link button on a Job

## Install (from Release)

Go to the releases tab and grab the latest release.

If you're on Firefox, it should just prompt you to install the extension when pressing on the .xpi.

If you're on Chromium, download the .zip and go to your extensions and enable developer mode. Then Load unpacked and put in the .zip.

## Local installation

Use make with your specific platform:

```sh
make chromium
make firefox
```

Or make for all platforms:

```sh
make all
```
